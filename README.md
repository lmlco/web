# lmlco/web

LifeMakersLabs（LML）アカウントの web 置き場。`index.html` は LML サイト（https://lmlco.github.io/ ）への誘導ページ。

## 履歴メモ（2026-07-01）
- ここにあった **土留め親ぐい計算ソフト（DKK／土木）** は、DKKアカウントへ移設済み → **https://dai1sep.github.io/soil-retaining-calc/** （repo: `dai1sep/soil-retaining-calc`）。
- LML と DKK のアカウント／思考を分離する方針に伴い、本リポジトリからは DKK コンテンツ（`chart.umd.min.js` / `jspdf.umd.min.js` / 旧 `index.html`）を撤去。
